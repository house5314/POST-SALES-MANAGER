# 공모전 코드 제출 패키지 생성 안내

## ZIP 만들기

```bash
npm run package:submission
```

생성 위치: `submission/POST-SALES-MANAGER-공모제출소스-YYYYMMDD.zip`

## 패키지에 포함되는 항목

- 애플리케이션 소스: `app/`, `components/`, `lib/`, `hooks/`, `types/`, `public/`
- 설정: `package.json`, `tsconfig.json`, `next.config.ts`, `Dockerfile`, `.env.example` 등
- 검증 스크립트: `scripts/`
- 문서: `개발-설계서.md`, `ai-internal-network.md`, `operations-data-quota.md`, `제출_README.md`
- `SUBMISSION_MANIFEST.json` (파일 목록)

## 제외 항목 (용량·보안)

- `node_modules/`, `.next/`, `.git/`
- `.env`, `.env.local` (비밀키)
- 내부 초안: `im.md`, `final.md`, `final2.md`
- `.cursor/`, `AGENTS.md` 등 개발 도구 메타

## 포털 제출 시 체크리스트

1. **개발 설계서** — `docs/개발-설계서.md` (PDF 변환 가능)
2. **소스 ZIP** — 위 명령으로 생성한 파일
3. **시연 URL** — Vercel 또는 자체 도메인
