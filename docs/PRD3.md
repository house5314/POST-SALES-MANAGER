아, 제가 범위를 너무 넓게 잡았군요! 메인 화면의 3단 구조와 UI는 이미 완벽하게 구축되어 있고, 오직 하단 [제안서 생성(인쇄/PDF)] 버튼을 눌렀을 때 출력되는 '결과물(문서)'의 내용과 구조를 개편하는 것이 목적이시군요.

메인 화면의 코드는 절대 건드리지 않고, 제안서 출력용 컴포넌트(또는 모달, 인쇄 화면)만을 방금 제가 제안해 드린 '데이터 기반 맞춤형 컨설팅 리포트' 구조로 완벽하게 덮어쓰도록 Cursor AI에게 지시하는 전용 PRD를 작성해 드립니다.

이 내용을 복사해서 Cursor 창에 넣고 "메인 화면 코드는 절대 수정하지 말고, [제안서 생성] 버튼을 눌렀을 때 나오는 제안서 컴포넌트의 내용만 이 PRD에 맞춰서 전문적인 구조로 재작성해 줘"라고 지시하시면 됩니다.

📄 [PRD] 생활정보홍보우편 맞춤형 영업 제안서(PDF/인쇄) 컴포넌트 고도화
1. Task Overview (작업 개요)
작업 목표: 사용자가 [제안서 생성(인쇄/PDF)] 버튼을 클릭했을 때 생성되는 문서(또는 모달, 인쇄용 화면)의 내용을 기존의 단순/범용적 문구에서 '공공데이터 기반의 맞춤형 컨설팅 리포트' 포맷으로 전면 개편한다.

제한 사항(Strict Constraint): 메인 화면의 레이아웃(60:20:20 구조), 행정동 검색, 업종 필터링, 지도 렌더링 로직 등 기존 메인 UI 코드는 절대 수정하지 않는다. 오직 제안서 출력용 컴포넌트(<ProposalDocument/> 등)만 수정한다.

2. Document Structure (제안서 시각적 구조 - A4 기준)
제안서는 인쇄 및 PDF 저장을 고려하여 깔끔하고 전문적인 B2B 리포트 형식으로 디자인되어야 한다. (@media print 또는 인쇄용 CSS 적용)

[Header] 제안서 제목 및 타겟 업체 기본 정보 (상호, 주소, 업종)

[Section 1: Data] 상권 데이터 진단 (API 데이터 바인딩)

[Section 2: Strategy] 맞춤형 홍보 전략 및 SWOT

[Section 3: ROI] 예상 ROI (투자 수익 시뮬레이션)

[Footer] 우체국 '생활정보홍보우편' 특장점 및 안내

3. Dynamic Template Logic (동적 렌더링 로직)
선택된 사업자(selectedBusiness)의 업종과 업력(설립연도) 데이터를 기준으로 3가지 템플릿 중 하나를 조건부 렌더링(Conditional Rendering)한다.

(Cursor AI는 아래의 텍스트를 컴포넌트 내에 정확히 맵핑해야 함)

📝 Template Type A: 외식업 / 기회 창출형 (신규 또는 성장 상권)
조건: 업종이 '외식업'이거나 상권 매출이 상위권일 때

상권 데이터 진단: "현재 해당 상권의 외식업 면적당 평균 매출은 지역 내 상위권으로 구매력이 매우 높으며, 배후에 풍부한 배달/포장 잠재 고객이 존재합니다."

맞춤형 홍보 전략 & SWOT:

전략: "주말 가족 단위 배달 수요를 선점하기 위해, 치킨/외식 소비가 극대화되는 '금요일 오후'에 각 가정에 우편물이 도달하도록 일정을 설계합니다."

SWOT 요약: S(풍부한 배후 수요), W(초기/동네 인지도 부족), O(배달/포장 수요 증가), T(전단지 등 로컬 광고비 상승)

예상 ROI 시뮬레이션: "우편물 {추천수량: 1,200}부 발송 시 (예상 비용: 약 00만 원) ➔ 보수적 전환율 1% (12건) 가정 시, 평균 객단가 기준 즉각적인 매출 증대 효과가 기대됩니다."

📝 Template Type B: 도소매업 / 위기 극복형 (귀금속, 의류 등 고관여 상품)
조건: 업종이 '도소매업'이거나 업력이 오래된 업체(5년 이상)일 때

상권 데이터 진단: "주변 상권의 오프라인 유동인구 변동 및 온라인 채널로의 고객 이탈 방어가 필요한 시점입니다. 고관여 상품 특성상 '신뢰도'와 '매장 방문' 유도가 가장 중요합니다."

맞춤형 홍보 전략 & SWOT:

전략: "무작위 전단지가 아닌, 고급스러운 카탈로그 형태의 우편물을 활용한 VIP 타겟팅 및 신뢰 마케팅을 제안합니다."

SWOT 요약: S(주거 밀집 상권), W(오프라인 방문율 하락), O(이벤트/명절 시즌), T(온라인 채널로의 이탈 가속)

예상 ROI 시뮬레이션: "우편물 {추천수량: 800}부 발송 시 (예상 비용: 약 00만 원) ➔ 고관여 상품 특성상 전환율 0.5% (4건) 가정 시, 건당 높은 객단가로 200만 원 이상의 매출 방어가 기대됩니다."

📝 Template Type C: 서비스업 / 생활밀착형 (미용, 잡화, 학원 등)
조건: 업종이 '서비스업' 등 도보 방문이 주력인 업체일 때

상권 데이터 진단: "주거 밀집 지역과 인접해 있어 도보 생활권 내 유동 인구가 매우 풍부합니다. 동네 이웃들에게 매장을 확실히 각인시킬 오프라인 접점이 필요합니다."

맞춤형 홍보 전략 & SWOT:

전략: "주민들의 매장 방문(Foot Traffic)을 직접 유도하기 위해, 우편물 지참 시 혜택을 제공하는 '실물 쿠폰 동봉형' 홍보를 제안합니다. 주말 방문 유도를 위해 목/금요일 도착을 권장합니다."

SWOT 요약: S(도보 생활권 내 위치), W(디지털 채널로 동네 타겟팅 한계), O(지역 내 소비 심리 회복), T(광고 채널 파편화)

예상 ROI 시뮬레이션: "우편물 {추천수량: 1,000}부 발송 시 (예상 비용: 약 00만 원) ➔ 쿠폰 회수율 2% (20건) 가정 시, 신규 단골 20명 확보 및 안정적인 추가 매출 창출이 기대됩니다."

4. Footer 공통 영역 (우체국 강점 강조)
모든 템플릿의 하단에는 다음 내용을 공통으로 고정 삽입한다.

우체국 '생활정보홍보우편'만의 독보적 강점:

100% 도달 보장: 전단지 무단 투기와 달리, 국가기관인 우체국 집배원이 각 세대 우편함에 직접 투함하여 도달률과 신뢰도를 극대화합니다.

예측 가능한 합리적 비용: 우체국을 통한 접수 시 최대 30%의 요금 감액 혜택을 적용받아 효율적인 마케팅 예산 운용이 가능합니다.

(안내구) 본 문서는 우체국 생활정보홍보우편 B2B 영업을 위한 맞춤형 컨설팅 데모 리포트입니다.

5. Cursor AI Action Directives
기존 메인 UI 컴포넌트(MapArea, TargetList, Filter 등)는 절대 건드리지 마라.

제안서 생성 버튼 클릭 시 렌더링 되는 인쇄용 컴포넌트 또는 모달 내부의 코드만 수정하라.

인쇄 시(@media print) 메인 화면의 지도나 필터 등의 불필요한 UI는 display: none 처리되고, 이 제안서 컴포넌트만 깔끔한 A4 비율 문서 형태로 인쇄되도록 CSS를 구성하라.

selectedBusiness 객체에서 업종(category) 데이터를 판별하여, 위 명세서의 Type A, Type B, Type C 중 적절한 텍스트 템플릿이 자동으로 맵핑되도록 분기문(if/switch)을 작성하라.
