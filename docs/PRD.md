PRD.md: 우체국 B2B 스마트 영업 네비게이터 (Post AI Smart Sales Navigator)1. 프로젝트 개요배경: 인사이동에 따른 영업 역량 편차를 해소하고, 데이터 기반의 과학적 영업을 통해 '생활정보홍보우편' 매출을 극대화함.목표: 매출 하락 업체(위기)와 배후 수요 증가(기회)를 매칭하여 최적의 B2B 잠재 고객을 발굴하고 자동화된 제안 도구를 제공함.  핵심 지표(KPI): 신규 계약 건수 증대, 제안서 작성 시간 단축, 영업 상담 표준화.2. 기술 스택 (Tech Stack)Framework: Next.js 14 (App Router)Styling: Tailwind CSS, Shadcn/ui데이터 소스: 공공데이터 상가·행정코드·국토부 등 API(공모전 제출용 별도 DB·Supabase 미사용)AI Engine: AI SDK (Google Gemini API - SWOT 분석 및 상담 멘트 생성용)Map: Kakao Maps API (위치 기반 시각화)PDF Generation: @react-pdf/renderer 또는 html2canvas & jspdf3. 데이터 구조 (참고 스키마 · 데모는 API 응답 직접 사용)3.1 businesses (사업자 정보)컬럼명타입설명idUUID기본키nameString업체명 (예: 범어동 스터디카페)categoryString업종 분류 (예: 독서실/스터디 카페)  addressString상세 주소 (대구 수성구 범어1동 등)  lat, lngFloat지도 표시용 좌표revenue_trendFloat전월 대비 매출 증감률 (예: -14.3)  3.2 market_stats (상권 통계)컬럼명타입설명region_codeString행정동 코드housing_countInt공동주택 세대수 (예: 12,630)  target_age_groupString주요 소비층 (예: 50대 여성)  peak_timeString매출 집중 시간대 (예: 18-23시)  floating_popInt일평균 유동인구  4. 화면 구성 및 UI 설계4.1 전체 레이아웃 (3-Column Layout)Left Section (지도): 카카오 지도 API를 활용해 관내 업체를 마커로 표시. 매출 하락 업체는 붉은색 마커로 강조.Center Section (상담 가이드 및 근거): AI가 생성한 표준 상담 멘트와 영업 선정 근거(SWOT)를 큰 카드로 배치.Right Section (필터 및 리스트): 주소(행정동), 업종, 매출 증감 범위를 필터링하는 사이드바.4.2 상세 컴포넌트 기능영업 선정 근거 카드 (Center):"범어1동 독서실은 매출이 14.3% 하락했으나, 주변 신축 아파트가 12,630세대나 유입되어 마케팅 시 즉각적인 효과가 예상됨"이라는 문구 자동 출력.  자동 상담 멘트 (Center):제시된 예시 스크립트를 기반으로 Gemini API가 업체별 맞춤형 문장을 완성하여 제시.제안서 생성 버튼: 클릭 시 SWOT 분석과 우체국 장점이 포함된 PDF 다운로드 기능.5. 핵심 기능 로직5.1 잠재 고객 발굴 알고리즘필터링: 매출 증감률 < 0 이고 주변 공동주택 증가율 > 10%인 업체를 '우선 영업 대상'으로 분류.  우편 발송 수량 추천: 해당 행정동 전체 세대수(8,560세대) 또는 인근 신축 아파트 세대수(12,630세대)를 AI가 추천 수량으로 제안.  5.2 AI 상담 멘트 생성 프롬프트 (Gemini)입력 데이터: 업체명, 매출 증감률, 주변 아파트 세대수, 주요 소비 연령대, 피크 타임.  출력 요구사항: "우체국은 100% 도달률과 30% 요금 할인을 보장합니다"라는 장점을 포함한 정중하고 신뢰감 있는 스크립트 작성.  5.3 PDF 제안서 포함 항목SWOT 분석: S(60개 학교 밀집), W(매출 하락), O(신축 단지 입주), T(신규 고객 편중).  배송 시점 추천: 유동인구가 많은 금요일 도착 권장.  민간 대비 강점: 집배원 직접 배달을 통한 100% 독취율 보장.
